Advanced Options - Array {{$index+1}}

Module Type

The module type describes the photovoltaic modules in the array. Default is standard panels.

Inverter Efficiency

The inverter's nominal rated DC-to-AC conversion efficiency. The default value is 96%.

Shading Losses

Reduction in the incident solar radiation from shadows caused by objects near the array such as buildings or trees. Default is 3%.

Annual Solar System Production Degradation

Solar panels degrade overtime depending on surrounding weather conditions, dust, etc. Typically this is covered in a manufacturer’s warranty to be less than 20% degradation over 25 years. The degradation set in the first array will be applied to all arrays. Default is 0.65%.
